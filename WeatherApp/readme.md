## üìù Project Overview

This is a responsive, single-page application (SPA) built to display current weather conditions and forecast data for any location searched by the user. The application focuses on a clean, modern design and user experience, adapting seamlessly across mobile and desktop viewports.
 
### Key Features

  * **Search Functionality:** Find weather data for cities globally.
  * **Current Conditions:** Displays temperature, "feels like," humidity, wind speed, and precipitation.
  * **Daily Forecast:** A seven-day outlook showing predicted high and low temperatures.
  * **Hourly Forecast:** A detailed, scrollable forecast for the selected day.
  * **Unit Conversion:** Allows users to switch between Metric and Imperial units for temperature, wind, and precipitation.
  * **Responsive Design:** Optimized for both narrow mobile screens (375px) and wide desktop viewports (1440px).

### Prerequisites

You only need a modern web browser to view this project.

### Installation and Setup

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
    ```
2.  **Navigate to the Project Folder:**
    ```bash
    cd YOUR_REPOSITORY_NAME/weather-app/
    ```
3.  **Open in Browser:**
    Open the `index.html` file directly in your web browser, or serve it using a local development server (like VS Code's Live Server extension).

## üé® Styling and Responsiveness

The application is designed to adapt its layout based on screen size:

| Viewport | Layout | Key CSS Techniques |
| :--- | :--- | :--- |
| **Mobile (max-width: 375px)** | Stacked vertical sections for clarity and scrolling. | Flexbox, `max-width`, and simplified components. |
| **Desktop (min-width: 1440px)** | Grid-based layout with two main columns for Current and Hourly Forecasts. | CSS Grid, `@media` queries, horizontal scrolling for Hourly data. |
